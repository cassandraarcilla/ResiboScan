## **How to Run**

1. Make sure Flutter is installed: [https://flutter.dev/docs/get-started/install](https://flutter.dev/docs/get-started/install)
2. Open a terminal in this folder
3. Run:

```bash
flutter pub get
flutter run
```

---

## **Features**

### **Milestone 1**

* Loading/splash screen with progress bar
* Home screen with 3 Cards using local AssetImage
* Card customization: elevation, margin, color, rounded corners
* Search bar with category filter chips
* Bottom navigation with FAB scan button
* Add receipt modal (manual)
* Folders screen (Personal / Work)
* Expense summary with breakdown bars
* Warranty alerts screen

---

### **Milestone 2**

* Multiple feature screens (Home, Folders, Expense Summary, Alerts)
* Navigation using `Navigator.push()` and `Navigator.pop()`
* Card-based UI used across multiple screens
* Local assets (icons/images) integrated in different pages
* Improved UI layout and screen composition

---

### **Milestone 3**

* Integration of RESTful API using `http` package
* Fetching data from API endpoint
* JSON data parsing with model class (`fromJson`)
* Display of API data in the app UI
* Loading state using `CircularProgressIndicator`
* Error handling using try/catch with error message UI
